# 🎵 Mood-Based Music Recommender

A Python app that detects your mood from text and recommends songs using Last.fm API.

## Features
- 50 mood categories
- 500+ curated songs
- NLP mood detection using TextBlob
- Last.fm API integration

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- Python
- Streamlit
- TextBlob
- Last.fm API
